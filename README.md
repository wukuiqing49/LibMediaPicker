 LibMediaPicker
 
 这是一个土拍你选择器的框架
 
   1:支持Gif
   
   2:支持视频
   
   3:支持图片预览放大
   
   4:支持视频外部预览
   

 
引用方式:

implementation  'com.github.wukuiqing49:LibMediaPicker:1.0.9'                 
   
 
 注意:需要处理 jdk 8 的兼容  
 
 compileOptions {                                              
        sourceCompatibility 1.8                                                    
        targetCompatibility 1.8                                                    
          }
